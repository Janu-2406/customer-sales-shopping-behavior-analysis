# Customer Sales & Shopping Behavior Analysis

## 📌 Project Overview

This project analyzes customer sales and shopping behavior using Python, SQL Server, and Power BI. The goal is to understand customer purchasing patterns, identify sales trends, and generate business insights that help businesses make data-driven decisions.

---

## 🎯 Project Objectives

- Analyze customer purchasing behavior.
- Identify sales trends across product categories.
- Perform data cleaning and exploratory data analysis (EDA).
- Answer business questions using SQL Server.
- Build an interactive Power BI dashboard.
- Provide business recommendations.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- SQL Server (SSMS)
- Power BI
- Jupyter Notebook

---

## 📂 Project Workflow

1. Data Cleaning using Python (Pandas)
2. Exploratory Data Analysis (EDA)
3. Business Analysis using SQL Server
4. Interactive Dashboard in Power BI
5. Business Recommendations

## 📈 Key Insights

- Clothing generated the highest revenue among all product categories.
- Most customers were non-subscribers.
- Young Adults contributed the highest revenue.
- Customer purchase behavior varied across different categories and age groups.
- Interactive filters allow users to explore data by gender, category, subscription status, and shipping type.

---

## 💡 Business Recommendations

- Focus on high-performing product categories.
- Reward loyal customers with offers and loyalty programs.
- Improve low-performing products through promotions.
- Plan marketing campaigns based on seasonal trends.
- Monitor customer feedback to improve satisfaction.
